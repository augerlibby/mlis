Goal 3: Navigate, curate and create information across the spectrum of human records.

Outcome 3a. Design resources or tools that facilitate access to information.

LIS 702: Facilitating User Learning and Information Needs

Reference Subject Guide: Women in Horror

I designed a reference guide using google sites as a project for LIS 702: Facilitating User Learning and Information Needs. I chose to highlight the topic “Women in Horror”, selecting reference books, books, databases, web resources, and video essays on the topic. This was an enjoyable project that allowed me to explore a topic interesting to me while designing a reference guide that will serve as an example in my future work as a librarian. This resource would allow students and those interested in the topic to further their research on the subject of women in horror.

Outcome 3b. Utilize resources or tools to manage information.

LIS 702: Facilitating User Learning and Information Needs

Instructional Presentation: Advanced Searching with the Rebecca Crown Library

For this outcome, I selected an instructional presentation I created for LIS 702: Facilitating User Learning and Information Needs. This presentation walks users through the process of advanced searching with the Rebecca Crown Library. I created learning outcomes to help guide this exercise, along with an outline of the steps taken in creating the presentation. I also reflected briefly on the exercise and how I might improve this presentation if I had more resources or time. This presentation meets this outcome because it demonstrates how to use a tool effectively, in order to access information through the Rebecca Crown Library.

Outcome 3c. Evaluate resources or tools that manage and facilitate access to information.

LIS 748: Collection Management

Selection Tool Reviews: Juvenile Fiction

In this assignment for LIS 748: Collection Management, I evaluated multiple juvenile fiction selection tools. I reviewed the following resources in the children’s literature realm: Children's Literature Comprehensive Database, Kirkus Reviews, Booklist and The Horn Book. I explored what features they provide and how these resources might benefit librarians and other users seeking access to book reviews. These resources help connect users to information, and in reviewing them, I successfully evaluated resources that manage and facilitate access to information.
